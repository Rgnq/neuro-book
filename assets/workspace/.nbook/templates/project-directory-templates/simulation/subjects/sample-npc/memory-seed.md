# Sample NPC Memory Seed

本文件只用于初始化 subject memory。创建正式 subject 时，把这里的初始化经历转换为 `events.jsonl`，把初始化稳定认知转换为 `memory.jsonl`；运行时不要反复读取本文件当作当前记忆。

## 初始化经历

- 示例 NPC 正在起始场景附近等待、观察或处理一件待定事务。
- 示例 NPC 能看见玩家角色出现在附近，但还没有确认对方身份、目的或是否可信。

## 初始化稳定认知

- `玩家角色`：玩家角色刚刚出现，我还没有确认对方身份、目的或是否可信。
- `起始场景`：我正在起始场景附近等待、观察或处理一件小事，能看见玩家角色。
